NAME
    ALTER DISTRIBUTION ADD DEFAULT SHARD - Adds a default shard to an existing
    distribution. Records that do not match any key range will be routed to this
    shard.

SYNTAX
    ALTER DISTRIBUTION <distributionID> ADD DEFAULT SHARD <shardID>

    where <shardID> is a default shard identifier. The <shardID> must exist.

DESCRIPTION
    Adds a default shard to an existing distribution. Records that do not match
    any key range will be routed to this shard.
