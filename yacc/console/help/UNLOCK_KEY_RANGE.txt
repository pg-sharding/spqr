NAME
    UNLOCK KEY RANGE - Releases the lock on a key range that was previously
    locked with LOCK KEY RANGE.

SYNTAX
    UNLOCK KEY RANGE <keyRangeID>

DESCRIPTION
    Releases the lock on a key range that was previously locked with LOCK KEY
    RANGE.
    After unlocking, queries routed to this key range will resume normal
    processing.

PARAMETERS
    <keyRangeID> - Identifier of the key range to unlock

TIPS
    - Always unlock key ranges after completing maintenance operations
    - Forgetting to unlock will cause queries routed to the key range to fail
      with an error

RELATED COMMANDS
    LOCK KEY RANGE
    SPLIT KEY RANGE
    UNITE KEY RANGE
    SHOW KEY RANGES
